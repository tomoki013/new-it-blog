## Next.js 15 が登場

ついに **Next.js 15** がリリースされました。
React 19 とのシームレスな統合が特徴で、特に **Server Actions** の進化が目覚ましいです。

```tsx
// React 19 の useFormState を使った例
import { useFormState } from "react-dom";

async function myAction(previousState, formData) {
  // ...
}

function MyComponent() {
  const [state, formAction] = useFormState(myAction, null);
  // ...
}
```

このブログも早速 Next.js 15 で構築しています。
