## なぜこのデザインなのか？

このブログの目的は、単なる備忘録ではなく「ポートフォリオ」です。
`GEMINI.md` にも記載した通り、デザインコンセプトは **「サイバーパンクとテック感の共存」** です。

### Velite による型安全なコンテンツ

記事の管理には `velite` を採用しました。
Markdown を型安全な JSON に変換し、TypeScript との相性は抜群です。

### ネオンアニメーション

`ArticleCard` の周りを回る光のアニメーションは、`conic-gradient` と `animate-spin` を組み合わせて実現しています。

```tsx
<div
  className="animate-[spin_8s_linear_infinite] group-hover:animate-[spin_4s_linear_infinite]"
  style={{
    backgroundImage:
      "conic-gradient(... oklch(var(--color-primary) / 0.8) ...)",
  }}
/>
```

技術とデザインの両方で「攻めた」ポートフォリオを目指します。

---

これらのファイルを作成・保存すると、`velite` が自動的に（`npm run dev` 実行中であれば）これらを検知してビルドします。

ブラウザで `http://localhost:3000` をリロードして、トップページに 3 つの記事カード（新しい順）が表示されるか確認してみてください。
