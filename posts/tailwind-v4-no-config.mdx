## Tailwind CSS v4 の衝撃

`tailwind.config.ts` が不要になったことは、単なる利便性の向上ではありません。
これは **Tailwind CSS v4** のアーキテクチャが根本的に変わったことを意味します。

### `@theme` ルール

すべての設定は `global.css` に集約されます。

```css
@import "tailwindcss";

@theme {
  darkmode: "class"; /* ここで設定 */

  colors: {
    primary: "oklch(var(--primary))";
  }
}
```

このアプローチにより、設定とスタイルが一元管理され、より直感的な開発が可能になりました。
